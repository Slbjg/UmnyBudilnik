## Описание проекта

Умный будильник  – это мобильное приложение для Android. Основная особенность приложения заключается в том, что выключить сигнал можно только с помощью трёх последовательных встряхиваний телефона. Такой механизм предотвращает случайное отключение будильника и способствует более быстрому пробуждению.

Приложение разработано на языке Kotlin с использованием Android SDK. В основе работы лежат компоненты AlarmManager для точного запуска по времени, Service для фонового воспроизведения сигнала и отслеживания движений, а также SensorManager для обработки данных акселерометра.

Приложение позволяет пользователю:

- установить точное время срабатывания будильника с помощью TimePicker;
- получить звуковой сигнал и вибрацию в заданное время;
- отслеживать прогресс встряхиваний через текстовый индикатор;
- выключить будильник только после трёх резких встряхиваний;
- видеть текущий статус работы будильника на главном экране.

## Основной сценарий работы приложения

1. Пользователь запускает приложение и видит экран с выбором времени.
2. С помощью TimePicker задаётся час и минута срабатывания.
3. После нажатия кнопки «Завести будильник» приложение устанавливает точный таймер через AlarmManager.
4. В назначенное время срабатывает AlarmReceiver, который запускает фоновый сервис AlarmService.
5. AlarmService воспроизводит циклический звуковой сигнал, включает вибрацию и начинает отслеживать показания акселерометра.
6. На экране отображается счётчик [0/3] и призыв встряхнуть телефон.
7. При каждом резком движении счётчик увеличивается, а устройство выдаёт короткую вибрацию.
8. После третьего встряхивания сигнал и вибрация прекращаются, сервис останавливается, а статус обновляется.

## Реализованный функционал

| Категория | Реализованные возможности |
|-----------|---------------------------|
| Установка времени | TimePicker с выбором часа и минуты |
| Точный запуск | AlarmManager с методом setExactAndAllowWhileIdle |
| Звуковой сигнал | MediaPlayer, системная мелодия будильника, зацикливание |
| Вибрация | Vibrator, пульсирующий паттерн на Android 8+ и legacy на старых версиях |
| Детектор встряхиваний | Акселерометр, вычисление резкости движения по скорости изменения координат |
| Обратная связь | Короткая вибрация при каждом засчитанном встряхивании |
| Фоновый режим | Service, работающий даже при выключенном экране |
| Адаптация под Android 12+ | Проверка разрешения SCHEDULE_EXACT_ALARM и запрос при необходимости |
| Счётчик | Текстовое отображение [0/3] – [3/3] |
| Статус | Текстовое поле с текущим состоянием будильника |

## Инструкция по установке и запуску

### 1. Копируйте репозиторий

```bash
git clone https://github.com/S1bjg/UmnyBudilnik.git
```

### 2. Откройте проект в Android Studio

- `File → Open` → выберите папку проекта
- Дождитесь синхронизации Gradle

### 3. Запустите приложение

Нажмите зелёную кнопку **Run ▶️**

> Важно: На эмуляторе акселерометр не работает. Встряхивание будет работать только на реальном телефоне.

### 4. Для Android 12 и выше

Если будильник не срабатывает:
- Настройки → Приложения → UmnyBudilnik → Разрешения → Точные будильники → Включить

## Технические параметры детектора встряхиваний

| Параметр | Значение | Назначение |
|----------|----------|------------|
| SHAKE_THRESHOLD | 100 | Порог чувствительности (меньше – легче засчитать движение) |
| SHAKE_COOLDOWN | 800 мс | Минимальный интервал между засчитываемыми встряхиваниями |
| STABILIZATION_DELAY | 2000 мс | Задержка после запуска сервиса для исключения ложных срабатываний |
